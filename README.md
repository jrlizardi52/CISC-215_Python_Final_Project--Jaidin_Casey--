## Intro/Problem Statement
The goal of this project is to develop a Number Guessing Game that challenges
users to correctly guess a randomly generated number within a user-defined range.
Through this project, students will practice:

- Control flow (loops and conditionals)

- Random number generation

- Input validation and output formatting

- Modular code organization

Git and GitHub for version control and submission
## Code Explanation
- Uses a function for the game
- random.randint() picks the number
- while loop runs guesses
- try/except handles bad input
- Main loop allows replay
